# Design System

- **Typography:** System UI, clean and modern
- **Colors:** Blue accents, neutral backgrounds
- **Components:**
  - Navbar: Top navigation, sticky, responsive
  - ProjectCard: Card layout, shadow, hover effect
  - ChatWithJulien: Placeholder box for chatbot
- **Spacing:** Consistent padding and margin using Tailwind utilities
- **Responsiveness:** Grid and flex layouts for all screen sizes
